# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest  | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please do **not** open a public GitHub issue.

Instead, report it privately via one of the following:

- **Email:** [rocm-ls@amd.com](mailto:rocm-ls@amd.com)
- **AMD PSIRT:** Follow AMD's responsible disclosure process at [https://www.amd.com/en/resources/product-security.html](https://www.amd.com/en/resources/product-security.html)

We will acknowledge receipt within 5 business days and provide a remediation timeline.

## Scope

This repository contains ROCm-LS demo and example code for AMD Instinct GPUs. Security issues related to
dependencies (ROCm, PyTorch, MONAI, etc.) should be reported to the respective upstream projects.
